# Random Password Generator

A simple, secure, and user-friendly password generator built with vanilla HTML, CSS, and JavaScript.

## Features

- Generate random passwords with customizable length (8-32 characters)
- Include/exclude uppercase letters, lowercase letters, numbers, and symbols
- Password strength indicator
- One-click copy to clipboard
- Responsive design
- Modern UI with gradient background

## How to Use

1. Open `index.html` in your web browser
2. Adjust the password length using the slider
3. Select which character types to include
4. Click "Generate Password"
5. Click the copy button to copy to clipboard

## File Structure

- `index.html` - Main HTML structure
- `styles.css` - Styling and layout
- `script.js` - Password generation logic
- `README.md` - Documentation

## Browser Compatibility

Works on all modern browsers (Chrome, Firefox, Safari, Edge)
